# python-everything_is_object

This project explores Python's object model in CPython: mutable vs.
immutable objects, references vs. copies, how variables are bound to
objects, and how arguments are passed to functions. It covers `id()` and
`type()`, object identity (`is`) vs. equality (`==`), integer and string
caching, tuples vs. lists, and writing a simple function to copy a list.

## Files

- `0-answer.txt` through `18-answer.txt`, `20-answer.txt` through
  `28-answer.txt` — short conceptual answers about object identity,
  equality, and mutability in Python.
- `19-copy_list.py` — a function that returns a copy of a list.
- `19-main.py` — test script used to verify `copy_list`.
