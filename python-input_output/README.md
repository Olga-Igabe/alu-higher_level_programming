# python-input_output

This project covers file I/O and JSON serialization in Python: reading and
writing text files with the `with` statement, converting Python objects to
and from JSON strings and files, serializing custom class instances, and a
classic Pascal's Triangle algorithm exercise.

## Files

- `0-read_file.py` — reads and prints a UTF8 text file
- `1-write_file.py` — writes a string to a text file, returns char count
- `2-append_write.py` — appends a string to a text file, returns char count
- `3-to_json_string.py` — returns the JSON string representation of an object
- `4-from_json_string.py` — returns an object from a JSON string
- `5-save_to_json_file.py` — writes an object to a file as JSON
- `6-load_from_json_file.py` — creates an object from a JSON file
- `7-add_item.py` — script that appends CLI args to `add_item.json`
- `8-class_to_json.py` — returns a dict description of an object's attributes
- `9-student.py` — `Student` class with a `to_json()` method
- `10-student.py` — `Student` with a filterable `to_json(attrs=None)`
- `11-student.py` — `Student` with `to_json()` and `reload_from_json()`
- `12-pascal_triangle.py` — builds Pascal's Triangle as a list of lists
